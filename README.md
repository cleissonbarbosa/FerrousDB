# FerrousDB
A Database write in rust, inspired by sqlite.

> [!CAUTION]
> This project is only for educational purposes, it is not recommended to use it in production.

- [x] Create a database
- [x] Create a table
- [x] Insert data
- [x] Select data
- [ ] Update data

- [ ] Transactions
- [ ] Joins
- [ ] Subqueries
- [ ] Aggregations
- [ ] Group by
- [ ] Order by
- [ ] Limit
- [ ] Offset
- [ ] Constraints
- [ ] Views
- [ ] Triggers
